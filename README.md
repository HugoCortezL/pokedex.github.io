<h2 id="running">Running the application 🎲</h2>
 
```bash
# Clone the repository
$ git clone https://github.com/HugoCortezL/pokedex.git
 
# Go to folder
$ cd pokedex
 
# Install the dependencies
$ npm install
 
# Start the application
$ npm run dev
```
